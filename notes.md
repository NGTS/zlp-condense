# Data sources

## Images

* `hjd` -> `header->tmid` + `hjd_correction`
* `flux` -> `Aper_flux_3`
* `fluxerr` -> `Aper_flux_3_err`
* `ccdx` -> `X_coordinate`
* `ccdy` -> `Y_coordinate`
* `skybkg` -> `Sky_level`
* `flux_[1-4]` -> `Aper_flux_[blah]`
* `error_[1-4]` -> `Aper_flux_[blah]_err`
